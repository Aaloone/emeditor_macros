## 排版脚本自由与分享版

本脚本用于辅助以EmEditor&reg;排版校对TXT或其他文本格式小说所用。基于“排版脚本中华太阳系联盟增强版”开发，根据个人使用时所遇到的问题增删修改了部分功能。基于互联网“我为人人，人人为我”的精神，在此共享。

## Typesetting script: freedom and sharing

The present script aims to asist typesetting chinese novel in TXT or other text format with EmEditor&reg;. The script is developed based on "solar system plus" version, with some functionality addition/deletion/modification based on my personal usage. I would like to share it here for carrying out the Internet spirit of freedom and sharing.
